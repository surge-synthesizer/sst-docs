Detailed Description

Adding a processor here.

Used to be we had a bunch of case staements. But now we are all driven by templates (see "processor.cpp" for some hairy template code you never need to touch). So to add a processor

  1. Add it in a .c/.hpp as a subtype of Processor

  2. Implement these constexpr values static constexpr const char *processorName{"OSC Pulse"}; static constexpr const char *processorStreamingName{"osc-pulse"}; static constexpr const char *processorDisplayGroup{"Generators"};

    The streaming name has to be stable across versions

  3. Specialize the id-to-type structure template <> struct ProcessorImplementor<ProcessorType::proct_osc_pulse_sync> { typedef OscPulseSync T; };

    and then include the h here and everything will work
