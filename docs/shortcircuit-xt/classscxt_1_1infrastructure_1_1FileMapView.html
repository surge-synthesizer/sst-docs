Detailed Description

This class provides a mechanism to do a MapViewOfFile/mmap in a portable way.

The intent is that you would use it with a unique_ptr where you needed the map scope.

fs::path p = string_to_path("foo.bin");
auto mapper = std::make_unique<scxt::FileMapView>(p);
auto d = mapper->data; // valid until mapper is destroyed
auto s = mapper->dataSize;
