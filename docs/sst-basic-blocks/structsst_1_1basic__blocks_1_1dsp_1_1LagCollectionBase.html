Detailed Description

template<int N, typename T>
struct sst::basic_blocks::dsp::LagCollectionBase< N, T >

LagCollectionBase is a bucket of laggers which act as a group and allow you to easily manage the active set.

Basically use it with <128> to smooth midi ccs and stuff like that

Nhow many laggers to have
