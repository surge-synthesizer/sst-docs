Detailed Description

template<typename DestinationHandler>
struct sst::basic_blocks::dsp::UIComponentLagHandlerBase< DestinationHandler >

Base class for lag handler assignment styles.

Basically DestinantionHandler needs to implement void setDestination(float) in whatever way it wants. Two versions (set direct value and set via method on reference) are below.
