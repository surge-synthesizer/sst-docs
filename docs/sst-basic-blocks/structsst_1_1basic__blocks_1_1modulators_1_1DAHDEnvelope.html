Detailed Description

template<typename SRProvider, int BLOCK_SIZE, typename RangeProvider = TenSecondRange>
struct sst::basic_blocks::modulators::DAHDEnvelope< SRProvider, BLOCK_SIZE, RangeProvider >

The ADSR or DAHD envelope provider.

Template Parameters
SRProvider- a class which provides two functions. ->samplerate is the current sample rate and envelope_rate_linear_nowrap which provides an envelope rate. The rate is BLOCK_SIZE * 2^-v / samplerate but is usually provided by a table lookup, like in SurgeStorage
BLOCK_SIZEthe block size
RangeProvider- sets mins and maxes
